# Traffic Accident & Weather Analytics Pipeline

This project investigates **how weather conditions influence road traffic accidents**, focusing on identifying patterns that explain why accidents occur more frequently in certain weather situations.

The solution implements a modern cloud-based analytics pipeline on AWS, integrating accident data with weather API data, transforming it, and enabling analytical queries to answer real business questions.
